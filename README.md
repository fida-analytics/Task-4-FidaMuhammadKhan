# Ecommerce Sales Dashboard - Power BI

## Project Overview
This project analyzes ecommerce sales data using Power BI.

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling

## Dashboard Features
- Total Orders KPI

- Sales Trend Analysis
- Orders by Payment Method
